## 🎉 토론 그룹 인라인 편집 기능 구현 완료!

### ✅ 구현된 기능들

#### **1단계: 기본 편집 시스템** ✅
- ✅ 편집 모드 토글 (편집/저장/취소)
- ✅ 멤버를 다른 그룹으로 이동
- ✅ 새 그룹 생성
- ✅ 그룹에서 멤버 제거 (미할당으로)
- ✅ API 연동 및 저장

#### **2단계: 고급 기능** ✅
- ✅ 모든 그룹 해제
- ✅ 시각적 피드백 (미할당 인원, 빈 그룹 표시)
- ✅ 빈 그룹 자동 제거
- ✅ 완전한 인라인 편집 UI

### 🎨 UI/UX 특징

#### **조회 모드**
- 👥 토론 조 정보
- [새로고침] [편집] 버튼

#### **편집 모드**
- 👥 토론 조 정보 (편집 중)
- [취소] [저장] 버튼

#### **미할당 인원 섹션**
- 📥 빨간색 테두리
- 각 멤버별 이동 버튼들: [→ 1조] [→ 2조] [→ 새 그룹]

#### **편집 가능한 그룹들**
- 🎯 각 조별로 구분
- 빈 그룹 표시
- 각 멤버별 이동 버튼들: [→ 다른조들] [그룹에서 제거]

#### **하단 액션 버튼**
- [새 그룹 추가] (파란색)
- [모든 그룹 해제] (빨간색)

### 🔧 핵심 기능

#### **멤버 이동**
```dart
void moveMemberToGroup(AttendanceIdAndNameModel member, int targetGroupNumber)
```

#### **새 그룹 생성**
```dart
void createNewGroup()
```

#### **저장**
```dart
Future<void> saveDiscussionGroupChanges(int meetingId)
```

### 🚀 사용 방법

1. **편집 시작**: 토론 조 정보 섹션에서 [편집] 버튼 클릭
2. **멤버 이동**: 각 멤버 옆의 [→ X조] 버튼 클릭
3. **새 그룹**: [→ 새 그룹] 또는 [새 그룹 추가] 버튼 클릭
4. **제거**: [그룹에서 제거] 버튼으로 미할당으로 이동
5. **저장**: [저장] 버튼으로 변경사항 API 전송
6. **취소**: [취소] 버튼으로 편집 취소

### ⚡ API 연동

#### **전체 그룹 구성**
```
PATCH /discussions/groups/assign?meetingId={meetingId}
Body: {
  "groups": [
    {"attendanceIdList": [1, 2, 3]},
    {"attendanceIdList": [4, 5]}
  ]
}
```

#### **개별 멤버 할당** (향후 확장용)
```
PATCH /discussions/groups/assign-member?meetingId={meetingId}&attendanceId={attendanceId}&groupNumber={groupNumber}
```
