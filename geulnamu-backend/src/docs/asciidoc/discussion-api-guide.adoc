== PATCH

=== 토론 그룹 구성 - 수동
* 접근 가능 권한 레벨: STAFF / 로그 생성
* 운영진 모임원이 토론 그룹을 구성하는 용도의 API
** 토론 그룹은 (현재) 최대 7개까지만 구성이 가능함
** 한 명의 모임원은 한 명의 그룹에만 할당 가능함
* 해당 기능에 의한 DB 수정의 경우, 'attendance' 테이블의 'lastModifiedAt' 컬럼에 기록되지 않음 (bulk update를 사용하기에)
