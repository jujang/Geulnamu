== Error Code List
|===
| Error Code | Description

| 400 - Bad Request Error | 요청값이 잘못된 경우 (아래의 경우에 해당하지 않는 모든 경우)
| 401 - Unauthorized Error | 토큰을 통한 보안 인가가 불가한 경우
| 403 - Forbidden Error | (토큰을 통한) 접근이 금지된 경우
| 404 - Not Found Error | 요청한 자원이 존재하지 않는 경우
| 417 - Expectation Failed | 요청값 중 유효성 검사를 실패한 값이 있는 경우
| 422 - Unprocessable Content | 요청한 처리를 진행할 수 없는 경우
|===
